Contribute!

Let's Build Awesome, Together

Brackets is an open source code editor built with JavaScript, HTML and CSS. It has been designed from the ground up to make it easy for web developers, like you, to hack on and extent its functionality.

Listed below, you'll find everything you need to get started.

Getting Started

Get the Source

So to start, you'll need Brackets, and you can find it on GitHub. Take a look a the code, fork it, pull a clone, whatever you want.

New to Git and GitHub? You can get started with github:help.
